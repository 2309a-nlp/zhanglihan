# 工单03 — 智能问答系统用户手册

## 快速开始

### 1. 启动系统

```bash
cd /path/to/工单03
python main.py
```

启动后自动预热连接池，浏览器自动打开 `http://localhost:8501`。

### 2. 首次使用

系统启动后：
- **侧边栏**显示系统状态（连接池、缓存、水印处理状态）
- **主区域**为对话界面
- 首次提问会建立 LLM 连接，耗时约 2-3s
- 后续问题利用连接池复用，耗时 < 2s

### 3. 提问

直接在输入框输入问题，支持中英文。
示例问题：
- "报告期内，武汉兴图新科电子股份有限公司来自军用领域的收入分别是多少？"
- "公司主营业务是什么？"
- "What is the company's main business?"

## 水印处理特性

系统会自动对 PDF 招股说明书进行水印检测和清洗：

1. **页眉/页脚清除**：自动识别并移除每页顶部的标题行（如 "武汉兴图新科电子股份有限公司 招股意向书"）
2. **背景水印清除**：检测并移除跨页重复的文本水印
3. **水印页过滤**：不包含实质文档内容的空白水印页会被自动跳过
4. **清洗透明**：侧边栏显示水印过滤状态和已移除的页数

如需关闭水印处理，修改 `config.py`：
```python
WATERMARK_REMOVAL_ENABLED = False
```

## 系统功能

### 侧边栏控制面板

| 区域 | 功能 |
|------|------|
| 🚀 速度优化 | 连接池状态、语义缓存条数、LLM tokens 数 |
| 💧 水印处理 | 水印过滤状态、已过滤页数 |
| 📚 文档管理 | 上传新 PDF、查看文档列表、删除文档 |
| ⚙️ 系统状态 | 文档数、向量数、模型信息 |
| 🗑 清空对话 | 清除当前对话历史 |
| 🔥 预热 | 手动触发连接池预热 |

### 对话功能

- 流式输出：边生成边显示，首字延迟 < 0.3s
- 来源标记：每条回答显示参考片段和相似度分数
- 计时显示：每次回答显示处理耗时（绿 < 3s、黄 3-6s、红 > 6s）
- 缓存标记：命中语义缓存时显示 💡 标记

## 常见问题

**Q: 为什么首次回答需要 5-6 秒？**
A: 首次需要建立 LLM API 连接（TCP 握手），后续利用连接池复用会显著加快。
点击侧边栏的「🔥 预热」按钮可预先建立连接。

**Q: 水印处理会影响检索准确率吗？**
A: 不会。水印处理只移除页眉/页脚/重复背景文字，保留实质文档内容。
相反，移除水印内容后检索噪声减少，准确率反而提升。

**Q: 如何上传新文档？**
A: 在侧边栏的文档管理区域，点击上传按钮选择 PDF 文件。
系统会自动解析、水印清洗、分块并建立索引。

**Q: 支持哪些语言？**
A: 支持中英文提问，中文文档检索效果最佳，英文关键词也可有效匹配。
